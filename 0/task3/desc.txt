The first line contains a number 'N' the length of the array and the number of queries.
The second line contains 'N' number between 0 and 1'000'000'000'000'000'000 this is the input array.
The third line contains 'N' number between 0 and 1'000'000'000'000'000'000 this are the queries.
The input array and the queries are also sorted.

Task :

print an integer for every query what is the closest number from the array. If there is multiple solution print the smallest answer.

Sample input:

5
1 6 20 80 100
2
3
4
9
90

Sample output
1
1
6
6
80
